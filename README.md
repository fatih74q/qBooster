# ⚡ qBooster v0.1 Beta ⚡

**qBooster** is a professional optimization tool with a **neon-themed interface**, designed to optimize Windows system resources, boost FPS in games, and clean up unnecessary system clutter.

## 🚀 Features
* **RAM Optimizer:** Clear unnecessary memory usage with a single click.
* **DNS Tuner:** Configure the fastest DNS addresses for better internet speed.
* **Junk Cleaner:** Remove temporary and useless system files (Temp, Prefetch, etc.).
* **CPU Priority:** Assign processor priority for games and high-performance apps.
* **Neon UI:** Modern, Full HD, and eye-friendly dark mode interface.

## 🛠️ How to Run?
You don't need to manually install libraries or deal with complex setups.

1.  Download this repository as a ZIP or use `git clone`.
2.  Navigate to the folder and double-click the **`Baslat.bat`** file.
3.  The launcher will automatically check for Python, install missing libraries, and start the app.

## 📋 Requirements
* Windows 10 / 11
* Python 3.10+ (The .bat file will notify you if it's missing)
